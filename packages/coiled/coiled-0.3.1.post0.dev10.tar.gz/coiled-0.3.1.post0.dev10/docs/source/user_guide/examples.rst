Examples
========

Dask, and therefore Coiled, are deeply integrated with the Python ecosystem. 

This list is by no means exhaustive (see `Dask Ecosystem <https://docs.dask.org/en/latest/ecosystem.html>`_), but here are some introductory examples of ways you can use Coiled to easily scale up popular tools you already use.
